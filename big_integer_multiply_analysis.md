# 大整数乘法分治算法分析

## 算法原理
Karatsuba算法是一种快速乘法算法，采用分治策略将大整数乘法分解为更小的子问题。基本思想是将两个n位数x和y分别分成两部分：

x = a×10^m + b  
y = c×10^m + d

其中m = n/2

传统乘法需要计算：
x×y = ac×10^(2m) + (ad+bc)×10^m + bd (需要4次乘法)

Karatsuba算法发现可以只用3次乘法：
x×y = ac×10^(2m) + [(a+b)(c+d)-ac-bd]×10^m + bd

## 时间复杂度分析
- 传统算法：O(n²)
- Karatsuba算法：O(n^log₂3) ≈ O(n^1.585)

递归关系式：T(n) = 3T(n/2) + O(n)

## 输入输出样例
输入：
x = 1234  
y = 5678

计算过程：
1. 分割：a=12, b=34; c=56, d=78
2. 计算：
   - ac = 12×56 = 672
   - bd = 34×78 = 2652
   - (a+b)(c+d) = 46×134 = 6164
   - 中间项 = 6164 - 672 - 2652 = 2840
3. 合并结果：6720000 + 284000 + 2652 = 7006652

## 优化改进方法
1. 混合算法：当数字较小时切换到传统算法
2. 多基数实现：使用更大的基数(如2^32)提高效率
3. 并行计算：三个乘法可以并行执行
4. 内存优化：减少递归过程中的内存分配
5. Toom-Cook算法：更高级的分治算法，可达到O(n^1.465)
